# enterprise-network-intrusion-intelligence-system
An AI-powered Enterprise Network Intrusion Intelligence System using Machine Learning to detect and classify malicious network traffic.

## Features

- Network Intrusion Detection
- Machine Learning-based Classification
- Data Preprocessing
- Model Training and Evaluation
- Interactive Dashboard (Streamlit)

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit
